# C++ 教程目录 {#tutorial_cpp_index}

@brief Fluxeem Driver C++ SDK 二次开发教程

本教程系列面向已经安装 Fluxeem Driver SDK 的用户，重点介绍如何基于 SDK 完成设备验证、基础接口调用、参数配置、RAW 文件读取和硬件同步演示。

## 目录

- @subpage quickstart - 快速开始
- @subpage camera_discovery - 设备发现与环境验证
- @subpage live_viewer - Live viewer与基础接口
- @subpage camera_config - Tool 接口与参数配置
- @subpage recording - File Reader 与 RAW 数据处理
- @subpage tool_demo - Tool 示例程序说明
- @subpage hardware_sync - 硬件同步与 HAL Sync

## 前置要求

- 已安装 Fluxeem Driver SDK
- Windows 或 Linux 运行环境
- 事件相机硬件和正确的 USB 3.0 连接方式
