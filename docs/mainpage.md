# Fluxeem Driver

**事件相机驱动与 SDK**

Fluxeem Driver 为事件相机（Event Camera）提供完整的软件接入方案，覆盖从 USB 设备发现、连接管理到事件流接收、参数配置和录制回放的全链路。适用于设备接入验证、算法原型开发、数据采集和上层应用集成。

## 面向读者

本文档面向使用 Fluxeem Driver SDK 进行二次开发的工程人员，重点内容：

- 事件相机的数据模型与基础操作方式
- 在工程中集成已安装的 Fluxeem Driver SDK
- 设备发现、相机连接、事件数据接收和参数配置
- 事件录制、RAW 文件回放和可视化
- 上层业务系统、算法模块和工具软件的接入参考

## 适用场景

| 场景 | 说明 |
| :--- | :--- |
| 设备接入 | 事件相机 USB 连接与功能验证 |
| 实时采集 | 事件流实时读取和可视化 |
| 离线分析 | 事件数据录制与 RAW 文件回放 |
| 算法开发 | 基于事件视觉的上层算法开发 |
| 系统集成 | 工程样机、研究原型和 SDK 嵌入 |

## 关键 API

| 类 | 职责 |
| :--- | :--- |
| `fluxeem::EvCameraService` | 设备发现、列举和打开 |
| `fluxeem::ICamera` | 相机生命周期、事件回调和工具访问 |
| `fluxeem::EvFileReader` | RAW 文件读取与回放 |
| `fluxeem::CameraTool` | 相机参数配置基类 |
