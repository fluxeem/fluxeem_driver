# 欢迎使用

欢迎使用 Fluxeem Driver。

Fluxeem Driver 是面向事件相机的驱动与 SDK，提供从设备发现、连接管理到事件流接收、参数配置、录制回放的一整套基础能力，适合用于设备接入、算法验证、数据采集和上层应用开发。

## 文档定位

本套文档面向使用 Fluxeem Driver SDK 进行二次开发的工程人员，重点说明如何基于已经打包好的 SDK 完成应用集成。

文档的主要目标：

- 帮助开发者快速理解事件相机的数据模型与基本使用方式
- 指导开发者在工程中集成已安装的 Fluxeem Driver SDK
- 说明如何发现设备、连接相机、接收事件数据并进行基础配置
- 演示如何进行事件录制、RAW 文件回放以及简单可视化
- 为上层业务系统、算法模块和工具软件提供稳定的接入参考

## 适用场景

Fluxeem Driver 适合以下类型的工作：

- 事件相机设备接入与验证
- 事件流实时采集和可视化
- 事件数据录制与离线分析
- 面向事件视觉算法的上层开发
- 工程样机、研究原型和内部 SDK 集成
